## Propositional Logic
- A statement that has a truth value.
  - The sky is blue.
- Propositional variables: p,q,r,s...
- Truth values: **T** for true, **F** for false.

## Compound Propostions

- Negation (not): &not;p
- Conjuction (and): p&and;q
